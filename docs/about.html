Intelligence Artificielle & Extraction

Prompting Avancé : Few-Shot + Chain-of-Thought

Les LLMs (Llama 3, GPT, Mistral) analysent les biographies Wikipédia via une technique de prompting sophistiqué :

1. Few-Shot Learning : Le modèle reçoit des exemples avant chaque requête (ex : "Isaac Barrow a enseigné à Newton → Influence directe").
2. Chain-of-Thought : Le LLM doit expliciter son raisonnement ("Mach a critiqué Newton, donc il a influencé Einstein").
3. Validation Multi-Critères : Chaque relation extraite est scorée par confiance (high/medium/low) puis validée via Wikidata SPARQL.

→ Résultat : Précision de ~85% sur les relations historiquement documentées.

Système Multi-LLM avec Fallback

Le code implémente une cascade de services pour garantir la disponibilité :

Cerebras → Groq (Llama 3) → Mistral → Ollama (local) → OpenAI

Si un service échoue, le suivant prend le relais automatiquement. Les réponses sont cachées avec versioning pour éviter les appels redondants.

Théorie des Graphes & Algorithmes

PageRank : L'Algorithme de Google

Chaque nœud (scientifique) reçoit un score d'influence récursive :

Intuition : Einstein est influent non seulement parce qu'il a influencé beaucoup de gens, mais parce qu'il a influencé des gens eux-mêmes influents.

→ La taille des nœuds dans la visualisation reflète ce score. Isaac Newton domine historiquement.

Algorithme de Louvain : Détection de Communautés

L'algorithme identifie automatiquement les "écoles de pensée" en regroupant les scientifiques densément connectés :

• ■ Physiciens quantiques (Bohr, Heisenberg, Pauli)
• ■ Mathématiciens des Lumières (Euler, Laplace, Gauss)
• ■ Philosophes grecs antiques (Platon, Aristote)

→ Les couleurs du graphe représentent ces clusters thématiques.

Betweenness Centrality : Les Ponts

Identifie les scientifiques qui connectent des domaines différents.

Exemple : René Descartes a un score élevé car il est à la fois :
→ Philosophe (lié à Spinoza, Leibniz)
→ Mathématicien (lié à Fermat, Pascal)

Ces "ponts" sont cruciaux pour la diffusion interdisciplinaire des idées.

Analyse Temporelle

Pondération par Proximité Historique

Les arêtes sont pondérées selon la différence d'époque entre deux scientifiques :

• Une influence entre contemporains (ex: Bohr → Heisenberg, 20 ans d'écart) est plus forte.
• Une influence posthume (ex: Euclide → Newton, 1800 ans) est plus faible.

→ Cela permet de distinguer les transmissions directes des influences conceptuelles lointaines.

Validation & Précision

Chaque relation extraite par l'IA est confrontée à Wikidata (base de connaissances structurée).

Le script validator.py interroge l'API SPARQL pour vérifier si la relation "X a influencé Y" est documentée.

→ Taux de confirmation : ~78% (les 22% restants sont des relations implicites non wikifiées mais historiquement plausibles).

Applications & Insights

Ce graphe n'est pas qu'une curiosité visuelle. Il permet :

  • Détection des "révolutionnaires" : via l'analyse des trous structurels (Burt's constraint), on identifie les scientifiques qui ont cassé des paradigmes.
  • Prédiction de liens manquants : l'algorithme Adamic-Adar suggère des influences probables non documentées.
  • Comparaison des traditions : analyse de densité entre écoles (Grèce antique vs Lumières vs Modernité).
  • Timeline interactive : filtre par époque (1400-2000) pour voir l'évolution du réseau.
